# iPad screenshots (App Store, 13-inch set source material)

Raw device screenshots captured 2026-07-20 by our iPad tester (David) on TestFlight build
**1.0.16 (9)** — the first iPad-capable build (`TARGETED_DEVICE_FAMILY = 1,2`, tablet layouts +
editor landscape mode).

Device class: 1536×2048 px (2048×1536 landscape), i.e. a 9.7-inch-class 4:3 iPad. These scale to
the App Store's required **iPad 13-inch** canvas (2064×2752 / 2752×2064) at exactly **1.34375×**
— same aspect ratio, no crop, no padding, no distortion. The store-ready upscales are derived
artifacts produced at upload time (not committed).
